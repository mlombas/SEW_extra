GeImage

Una aplicación de intercambio de fotografías

Motivación

La fotografía es un hobby (o profesión) apasionante. Esta aplicación fué creada con el objetivo de que cualquier usuario pueda ver fotografías tomadas cerca del lugar en el que está, de forma que pueda ver paisajes o puntos destacados con facilidad.

Capabilidades básicas

Tienes una foto que has sacado y quieres subirla? ¡Perfecto! Solo tienes que dirigirte a la página de compartir. Ahí encontrarás varias opciones para subirla. Puedes subirla seleccionando todos sus metadatos, o, si prefieres, ¡puedes subir un XML directamente con todo!
